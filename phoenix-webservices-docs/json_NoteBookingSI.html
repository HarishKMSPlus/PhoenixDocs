NoteBookingSI Data Type

Result of getting Bill of Lading data with the Shipping Instruction for Note Booking

Properties
name data type description
blExist boolean To verify Bill of Lading already create or not
countSI number Count the Shipping Instructions uploaded in Booking
dateTimeBLCreateWithSI string To get version of the Shipping Instructions which create with Bill of Lading
dateTimeSelectedSI string To get version of the Shipping Instructions which selected from Note Booking
versionCheck string To get result of comparison between the Shipping Instructions which create with Bill of Lading and the selected from Note Booking
createWithSI boolean Verify Bill of Lading create with the Shipping Instructions
latestSISelected boolean Verify the selected Shipping Instruction is latest
dateTimeBLCreateManually string To get date/time Bill of Lading created/updated
